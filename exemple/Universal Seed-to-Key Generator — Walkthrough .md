# Universal Seed-to-Key Generator — Walkthrough

## What Was Done

Successfully extracted and decrypted the complete Voltage lookup table, generating all 65536 seed-to-key pairs. Integrated this database directly into the Android application to enable instant, automatic generation of security keys for HV Battery unlocking.

## How it works

1. **The Database:** The `decrypted_table.txt` string (containing 65536 exact keys separated by `L`) is stored in `res/raw/keys.txt`. This file size is only around ~320KB, making it extremely lightweight for an Android app while having zero loading latency.
2. **The Generator:** `SeedKeyGenerator.kt` loads the resource on initialization. When a seed is received (e.g., `0000`), it converts the hex to an integer index `0`, and pulls the key at that exact array position (`02CC`).
3. **The User Flow:** 
   - When the user starts the HV Battery unlock sequence, the ELM327 queries the ECU.
   - The ECU returns a 4-character seed.
   - The app matches the seed using `SeedKeyGenerator.generateKey(seed)`.
   - The UI Key Input dialog appears with the generated key already pre-populated, and a green message saying: *"Universal generator found key:"*
   - The user just presses **SUBMIT** without needing to type anything!
